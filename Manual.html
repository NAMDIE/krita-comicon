Krita Comic Creator - User Manual Table of Contents Installation Getting Started Project Management Panel Layouts Speech Bubbles SFX Library Image Import Exporting Keyboard Shortcuts Troubleshooting Installation Download the latest release from GitHub Extract the comic_creator folder to your Krita pykrita directory: Linux: ~/.local/share/krita/pykrita/ Windows: %APPDATA%\krita\pykrita\ macOS: ~/Library/Application Support/krita/pykrita/ Restart Krita Go to Settings → Configure Krita → Python Plugin Manager Enable Comic Creator Restart Krita again Access the docker: Settings → Dockers → Comic Creator Getting Started Creating Your First Comic Go to File → New Comic Project Fill in project details (Title, Author, Series, Issue) Select page format (US Standard, Manga, European, etc.) Set default gutter and margin sizes Click Create Project Quick Start: The plugin creates a first page automatically. Use the Panels tab to apply a template and start drawing! Project Management Adding Pages In the Pages tab, click Add Page. Select a template or start with a blank page. Managing Pages Duplicate: Create a copy of the selected page Delete: Remove a page from your project Reorder: Drag pages to rearrange sequence Tip: Page thumbnails update automatically. Click any thumbnail to jump to that page. Panel Layouts Using Templates The Panels tab contains 85+ preset templates: Standard Grids: 2×3, 3×2, 3×3, etc. Western Comics: Hero moments, action sequences Manga: Vertical flow, diagonal cuts European BD: Balanced, architectural layouts Splash Pages: Full page or double-spread Custom Layouts Click Custom Template button Opens the Template Editor Add panels using rectangle, circle, or custom shape tools Adjust positions by dragging or entering precise values Save as new template for reuse Panel Properties Border Width: Thickness of panel borders Gutter Size: Space between panels Clip Content: Artwork stays within panel boundaries Speech Bubbles Available Types Standard: Regular dialogue Thought: Cloud-style for internal thoughts Shout: Jagged edges for yelling Whisper: Dashed border for quiet speech Radio/Electronic: Zigzag border Narration: Rectangular caption boxes Adding Bubbles Go to Assets tab Select bubble type from Speech Bubbles section Click in your panel to place Drag to resize Adjust tail to point at character Add text using text tool SFX Library Categories Impact: POW, BAM, CRASH, THUD, SMASH Motion: ZOOM, WHOOSH, SWING, DASH Sound: RING, BEEP, BANG, CLICK Emotional: GASP, SOB, SIGH Using SFX Select category from dropdown Click SFX button to add to active panel Position and scale as needed Customize color and style in layer properties Image Import Import to Panel Select a panel by clicking it Click Import to Panel Select image file Image is automatically clipped to panel boundaries Paste from Clipboard Copy image to clipboard (from browser, photo editor, etc.) Select target panel Click Paste from Clipboard Image appears in panel, clipped to boundaries Tip: Use imported photos as reference layers (reduce opacity) to trace or for photo-realistic backgrounds. Exporting Export Formats PNG: Lossless, supports transparency JPEG: Smaller file size, good for web PDF: Print-ready with embedded pages CBZ: Comic book archive format PSD: Preserve layers for further editing Export Process Go to File → Export Comic Select output directory Choose format Select page range (All, Current, or Custom) Set DPI (300 for print, 72-150 for web) Enable options: Flatten Layers Include Bleed Marks Click Export Keyboard Shortcuts ActionShortcutNew PageCtrl + NNext PagePage DownPrevious PagePage UpAdd PanelCtrl + PAdd Speech BubbleCtrl + BExport ComicCtrl + E Note: Shortcuts can be customized in Krita's settings Troubleshooting Plugin Not Appearing Check plugin is in correct directory Verify it's enabled in Python Plugin Manager Restart Krita twice (required for new plugins) Check Krita version is 5.0 or higher Templates Not Loading Check data/templates/ folder exists Verify JSON syntax in template files Clear template cache and reload Export Issues Ensure output directory has write permissions Check available disk space For PDF: ensure document is saved first For CBZ: verify images exported successfully Performance Large projects: Export pages individually Reduce canvas size for web comics Flatten unused layers Merge completed panels Getting Help If problems persist: Check GitHub Issues for known problems Report bugs with Krita version, OS, and error logs Join community discussions Credits Krita Comic Creator is open source software developed by the community.
