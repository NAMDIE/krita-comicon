# Krita Comic Creator

A comprehensive plugin for Krita enabling professional multipage comic book and graphic novel creation with advanced panel layouts, speech bubbles, SFX libraries, and export capabilities.

## Features

- **Multi-page Project Management**: Create and manage comic projects with metadata
- **85+ Panel Templates**: Industry-standard layouts from Western, Manga, European BD traditions
- **Visual Template Editor**: Create custom panel layouts with drag-and-drop interface
- **Speech Bubble Library**: Extensive preset collection (standard, thought, shout, whisper, etc.)
- **SFX Graphics Library**: Impact, motion, and sound effect graphics
- **Image Import to Panels**: Paste or import images directly into panel boundaries
- **Professional Export**: PDF, CBZ, individual pages with print-ready options
- **Non-destructive Workflow**: Vector-based panel boundaries

## Installation

1. Download the latest release
2. Extract to your Krita pykrita directory:
- **Linux**: `~/.local/share/krita/pykrita/`
- **Windows**: `%APPDATA%\\krita\\pykrita\\`
- **macOS**: `~/Library/Application Support/krita/pykrita/`
3. Restart Krita
4. Enable plugin: Settings → Configure Krita → Python Plugin Manager → Comic Creator
5. Access via: Settings → Dockers → Comic Creator

## Quick Start

1. Create new comic project: File → New Comic Project
2. Select page template or create custom layout
3. Draw in panels (automatic clipping)
4. Add speech bubbles and SFX from library
5. Export: File → Export Comic

## Requirements

- Krita 5.0 or higher
- Python 3.6+
